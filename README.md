# OC Character Website

这是一个用于展示原创角色（OC）设定的静态网页项目，适合部署到 GitHub Pages 作为个人角色站点。

项目当前包含一个简单的前端首页，用来展示：

- 角色基础档案
- 故事背景和时间线
- 印象关键词和氛围内容
- 作者备注

## 文件结构

- `index.html`：页面主体结构
- `style.css`：页面样式
- `script.js`：简单的前端交互

## 使用方式

直接修改 `index.html` 中的角色文字内容，并根据需要替换样式或图片内容即可。

## 部署

将项目上传到 GitHub 仓库后，可以通过 GitHub Pages 直接发布为静态网站。
